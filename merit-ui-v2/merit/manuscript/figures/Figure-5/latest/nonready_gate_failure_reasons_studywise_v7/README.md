# Non-Ready Gate Failure Reasons, Studywise v7

Source cache: `merit-cache-workbench-full-v7`

Scope: study-level records with final band other than Ready (`n=2,657`).

Best visualization: a two-panel summary. Panel A assigns one conservative primary limiting reason per study; Panel B counts all non-pass gates because G2/G4/G3/G5 often co-occur.

Main result: the dominant hard gate failure is G4 minimum class support: 987/2,657 non-Ready studies (37.1%). Including warnings, G4 is non-pass in 2,072/2,657 studies (78.0%).
